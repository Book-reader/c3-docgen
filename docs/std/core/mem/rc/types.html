std::core::mem::rc

Public

RefCounted

A RefCounted struct should be an inline base of a struct.If a `dealloc` is defined, then it will be called rather than `free`For convenience, a ref count of 0 is still valid, and the struct isonly freed when when ref count drops to -1.The macros rc::retain and rc::release must be used on the full pointer,not on the RefCounted substruct.So `Foo* f = ...; RefCounted* rc = f; rc::release(rc);` will not do the right thing.
struct RefCounted
{
	Atomic{int} refcount;
}
